# 更新日志

所有显著变更都将记录在此文件中。

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-11-07

### 新增功能
- ✨ 实现了完整的 RESTful API (提交、查询、取消作业)
- ✨ FIFO + First Fit 资源调度算法
- ✨ 异步 API 服务（FastAPI + asyncpg）
- ✨ 同步 Worker 服务（RQ + psycopg2）
- ✨ PostgreSQL 持久化存储
- ✨ Redis 任务队列
- ✨ 完整的 Docker 容器化支持
- ✨ Loguru 结构化日志
- ✨ 健康检查和清理脚本
- ✨ 完整的文档（README、架构、API 示例、部署指南）

### 技术特性
- 🔧 线程安全的资源跟踪器
- 🔧 异步日志文件读取
- 🔧 作业超时控制
- 🔧 优雅关闭机制
- 🔧 数据库连接池管理
- 🔧 幂等的取消操作

## 贡献指南

### 版本号规范

- **主版本号 (MAJOR)**: 不兼容的 API 变更
- **次版本号 (MINOR)**: 向后兼容的功能新增
- **修订号 (PATCH)**: 向后兼容的问题修复

### 更新日志格式

每个版本包括以下部分：

- **新增功能 (Added)**: 新功能
- **变更 (Changed)**: 现有功能的变更
- **废弃 (Deprecated)**: 即将废弃的功能
- **移除 (Removed)**: 已移除的功能
- **修复 (Fixed)**: 问题修复
- **安全 (Security)**: 安全相关的变更
