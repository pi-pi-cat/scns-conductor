## 🚀 使用新导航系统

### 入口选择

```
┌─────────────────────────────────────┐
│   我想...                           │
├─────────────────────────────────────┤
│ 全面了解文档         → INDEX.md     │
│ 快速开始使用         → README.md    │
│ 查看目录结构         → DIRECTORY_TREE.md │
│ 直接使用API          → API_EXAMPLES.md  │
│ 部署系统             → DEPLOYMENT.md    │
└─────────────────────────────────────┘
```

### 查找文档的三种方式

#### 方式 1: 按主题查找（INDEX.md）

```
快速开始 → API_EXAMPLES.md
架构设计 → ARCHITECTURE.md
容错机制 → FAULT_TOLERANCE_SUMMARY.md
```

#### 方式 2: 按场景查找（README.md）

```
我想部署系统 → DEPLOYMENT.md
我想调用API → API_EXAMPLES.md
我想理解架构 → ARCHITECTURE.md
```

#### 方式 3: 按文件名查找（DIRECTORY_TREE.md）

```
按字母顺序索引
A → ARCHITECTURE.md
D → DEPLOYMENT.md
W → WORKER_CONCURRENCY.md
```

## 🔧 维护建议
